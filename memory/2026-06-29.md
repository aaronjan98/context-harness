## Markdown import checkpoint
- Implemented backend Markdown import at
  `POST /api/conversations/{conversation_id}/imports/markdown`.
- Import supports three light patterns:
  Markdown speaker headings such as `## User` / `## Claude`;
  prefixed speaker lines such as `User: ...` / `Assistant: ...`;
  paragraph fallback where each paragraph becomes a user message.
- Import writes canonical `messages/mNNNN.md` files, updates conversation
  metadata, refreshes `exports/current.md`, and does not implicitly create
  missing conversations.
- This checkpoint supports the product premise by allowing copied web chatbot
  or external agent transcripts to become Context Forge source-of-truth
  conversations before later AI adapters continue them.
